About

A project to analyse Bach Chorales. Currently, it can check the presence of parallel 5ths and 8ths and identify the Pitch Class Set of the chords. Future implementations of possibly: detection of cadences, generation of Bach Chorales. How it works: At every instance when there are 4 'on' note events, a 'chord' is formed. These chords will then be passed into the FourParts package and analysed. When uploading the midi file, ensure that is has strictly only 4 parts.
